Keep Event Organiser plugin Events in sync with CiviCRM Events.



== Description ==

A WordPress plugin for syncing Event Organiser Events, Venues and Event Categories to their corresponding entities in CiviCRM.

This plugin requires at least WordPress 3.6 and CiviCRM 4.6.

It also requires:

* Event Organiser version 3.0 or greater
* Radio Buttons for Taxonomies to ensure only one event type is selected

Be aware that this plugin is still in development. Test often, test thoroughly and open an issue if you find a problem.



== Installation ==

1. Extract the plugin archive
1. Upload plugin files to your `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
